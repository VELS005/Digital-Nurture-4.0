# Bootstrap Folder Structure

- **css/**  
  Contains Bootstrap's compiled CSS files (e.g., `bootstrap.min.css`).  
  Use these to style your HTML pages.

- **js/**  
  Contains Bootstrap's JavaScript files (e.g., `bootstrap.bundle.min.js`).  
  These enable interactive components like modals, dropdowns, and tooltips.

- **icons/**  
  (If present) Contains Bootstrap Icons SVGs or font files.  
  Use these for scalable vector icons in your UI.
